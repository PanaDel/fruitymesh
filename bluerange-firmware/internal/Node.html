Terminal Commands

Setting the Serial Number

set_serial [serial number]

Sets the serial number of the connected beacon. The serial number is stored in persistent data and thus will be the same on reboot.

Examples

set_serial BRTCR

Setting the node key

set_node_key [hex- or base64-string]

Sets the node key of the connected beacon. The node key is stored in persistent data and thus will be the same on reboot.

Examples

set_node_key 00:11:22:33:44:55:66:77:88:99:AA:BB:CC:DD:EE:FF
