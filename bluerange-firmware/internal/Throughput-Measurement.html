Throughput Measurement

Measurement Technique

Throughput is measured by starting a timer as soon as we start sending messages and stop the timer when we receive the response of all messages being sent.

Measurement Readings

Measurement was done by distributing beacons both widely (approx. 5-6 m apart) and closely (approx 1-2m apart). PingPong command from DebugModule was used for measuring the throughput. Detailed description of pingpong command can be found in DebugModule.adoc. The following table describes different measurement scenarios:
